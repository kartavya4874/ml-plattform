# Parametrix AI Backend

This is the FastAPI backend for the Parametrix AI platform, hosted on Hugging Face Spaces.

## Deployment Details
- **Framework:** FastAPI
- **SDK:** Docker
- **Hardware:** CPU Upgrade (Recommended)

## Environment Variables Required
Make sure to set the following in your Space Settings:
- `MONGODB_URL`
- `RESEND_API_KEY`
- `R2_ACCESS_KEY_ID`
- `R2_SECRET_ACCESS_KEY`
- `JWT_SECRET_KEY`
- etc.
